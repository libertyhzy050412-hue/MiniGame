# 锈甲长歌 · Armor Stained With Sorrow — 系统配置要求

## 可执行文件路径

| 类型 | 相对路径 |
|------|----------|
| 安装版 | `release/锈甲长歌 Setup 0.0.0.exe` |
| 免安装版 | `release/win-unpacked/锈甲长歌.exe` |

---

## 系统配置

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| **操作系统** | Windows 10（64 位） | Windows 10 / 11（64 位） |
| **处理器** | Intel Core i3-2100 或 AMD 同级 | Intel Core i5-8400 或 AMD 同级 |
| **内存** | 4 GB RAM | 8 GB RAM |
| **显卡** | Intel HD Graphics 4000（集成显卡） | NVIDIA GeForce GTX 750 Ti 或同级（独立显卡） |
| **存储空间** | 500 MB 可用空间 | 1 GB 可用空间（SSD 推荐） |
| **DirectX 版本** | DirectX 11 | DirectX 12 |
| **声卡** | 兼容 DirectSound | 兼容 DirectSound |

---

> 游戏基于 Electron 42 + Vite 构建，实际对硬件要求较低。最低配置即可流畅运行 1920×1080 分辨率。
